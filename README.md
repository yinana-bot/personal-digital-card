# personal-digital-card

## 作品名称
个人数字名片页

## 作品简介
这是一个个人数字名片网页，采用现代化的卡片式设计，展示个人信息、联系方式、自我介绍、兴趣爱好和技能标签等内容。页面具有响应式布局，适配电脑和手机设备，并包含多种交互功能提升用户体验。

## 页面主要内容
- **头部区域**：个人头像、姓名、学号、专业、个性签名
- **动态问候**：根据时间段显示不同问候语（早上好/下午好/晚上好）
- **联系方式**：邮箱、电话、GitHub外部链接
- **自我介绍**：约180字的个人介绍
- **兴趣爱好标签**：6个兴趣标签，点击弹出对应图片
- **技能标签**：3个技能标签，带进度条显示掌握程度
- **深色模式切换**：点击右上角按钮切换浅色/深色主题
- **背景音乐**：右下角按钮控制播放/暂停
- **底部版权信息**

## AI辅助开发说明

### 是否使用AI
是

### 使用的工具
- DeepSeek（对话式AI辅助）
- ChatGPT（代码结构参考）

### 关键提示词
1. "请帮我设计一个个人数字名片页的HTML结构，包含头像、姓名、学号、专业、自我介绍、兴趣标签和技能标签"
2. "如何实现响应式布局，让卡片在手机上也能正常显示？"
3. "请提供CSS样式，让页面看起来现代简洁，使用渐变背景和圆角设计"
4. "添加技能进度条，滚动到可视区域时自动填充动画"
5. "点击兴趣标签弹出对应图片的弹窗功能"
6. "添加背景音乐控制按钮，支持播放/暂停"
7. "添加动态时间问候，根据时间段显示不同问候语"

### AI主要帮助的部分
1. HTML结构框架设计
2. CSS样式代码生成（渐变背景、圆角卡片、响应式布局）
3. 深色模式切换功能实现
4. 技能进度条及滚动动画
5. 弹窗功能实现
6. 背景音乐控制逻辑
7. 动态时间问候函数
8. 自我介绍文本润色
9. 文件结构规划建议

### 对AI结果做的修改
1. 调整了颜色方案，使用青蓝色系替代紫色
2. 添加了背景五角星效果（SVG星星，提升视觉氛围）
3. 优化了弹窗关闭按钮样式，使其在圆形中居中
4. 调整了技能进度条颜色和动画时长
5. 将自我介绍改为中性客观的表述
6. 删除了部分冗余CSS代码
7. 统一了圆角和阴影风格
8. 根据实际需求移除了鼠标跟随星星效果（优化性能）

### AI最不可靠的一点
AI生成的颜色方案有时不符合个人审美，需要手动调整；自我介绍文本偏向夸张叙述的风格，与实际情况不符，需要重新修改；性能优化方面AI不会主动考虑，高频交互效果（如鼠标跟随）可能导致页面卡顿，需要人工评估取舍；代码兼容性方面可能遗漏某些浏览器的适配写法。
